# READ ME

Use this action to install Tuyen CLI.

Documentation: <https://cli.tuyen.blog/>.

## Installation

```yaml
steps:
  - name: Setup Tuyen CLI
    uses: tuyen-at-work/tuyen-cli-action@master
```
